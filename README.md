# Rock Paper Scissors Game ✊✋✌️

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features
- Play against the computer
- Random computer moves
- Score tracking (Win, Lose, Tie)
- Score saved using Local Storage
- Reset button to clear score

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Local Storage API

## 🎮 How to Play
1. Click on Rock, Paper, or Scissors
2. The computer will randomly choose its move
3. The result will be displayed
4. Your score will update automatically

## 🧠 How It Works
- `Math.random()` is used to generate computer moves
- Game logic compares player and computer choices
- Results and score are displayed dynamically
- Scores are saved using `localStorage`
